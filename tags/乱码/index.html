URL中的中文出现乱码的解决方法

很多情况下，我们需要在URL里传递中文参数。网上也很多解决方法，我在这里把解决方法列一下，并加上点自己遇到的问题。
这里主要是Java工程。

需要做的是：
1.编码
2.解码

1.编码

Java

String encodeStr = java.net.URLEncoder.encode("中文","UTF-8");

Javasript

var encodeStr = encodeURI(encodeURI("中文"));

2.解码

如果URL为 var url = http://xxxx.com?name=+encodeURI(encodeURI("中文"))
Java

Stirng name=request.getParameter("name");
name=new String(name.getBytes("ISO-8859-1"),"UTF-8");//方法1，推荐使用该方法
name=URLDecoder.decode(name,"utf-8");//方法2，该方法有可能会解析不了
