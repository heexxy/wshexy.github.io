关于Hexo + Github 搭建个人博客

By 何潇洋

Published Feb 12 2015

一直都想搭建一个个人博客，但是总是因为各种懒没有付出实际行动。偶尔发现了Hexo这个好东西，就自己搭建了一个。

具体如何搭建，网上都有很详细的教程。我只说我遇到的一些问题。
1.第一步一定要安装node.js，最新版本就好。
2.在根据教程一步一步到最后，启动服务会发现自己的页面全是代码。这是因为缺少几个插件造成的，安装命令如下：
1.npm install hexo-renderer-ejs
2.npm install hexo-renderer-stylus
3.npm install hexo-renderer-marked
